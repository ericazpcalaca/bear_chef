# BC Bear Chefs

A full-stack web platform that connects culinary talent with individuals seeking fine dining experiences — a marketplace where users can book private chefs for meals, cooking classes, and special events.

Built as a group project (5 members) using **PHP**, **MySQL**, and **CSS**, running on an Apache/XAMPP stack.

---

## About the Project

The culinary world is often seen as exclusive and hard to access — aspiring chefs struggle to gain practical experience, while people who want a fine dining experience at home often can't find a way to connect with talented chefs. **BC Bear Chefs** bridges that gap with an online platform where:

- **Customers** can browse chef profiles and menus, place orders, specify dietary preferences, schedule meals, and rate/review chefs.
- **Chefs** can manage their profile and availability, receive and respond to booking requests, and track their order history.

The project began with a formal business case, including a mission statement, revenue model, and a full set of functional and non-functional requirements, before moving into development.

## Features

**For Customers**
- Account registration and secure login
- Profile management (personal info, dietary preferences)
- Browse chefs and menus
- Place and schedule orders
- Track order status
- Rate and review chefs
- Send notifications to chefs

**For Chefs**
- Account registration and secure login
- Profile management (specialties, experience)
- View, accept, or decline incoming meal requests
- Mark requests as completed
- View order history
- Manage availability and menu offerings

## 🛠️ Tech Stack

- **Backend:** PHP

## Getting Started

### Prerequisites
- [XAMPP](https://www.apachefriends.org/) (or any Apache + MySQL + PHP environment)
- A web browser

### Installation
1. Clone the repository into your XAMPP `htdocs` folder:
   ```bash
   git clone https://github.com/ericazpcalaca/bear_chef.git
   ```
2. Start Apache and MySQL from the XAMPP control panel.

   <img src="https://github.com/DantteSousa/Group4Project/assets/15451346/2df70b82-caa6-413e-ba3b-32987f172505" alt="XAMPP control panel" width="400">

3. Create a database called `bc_database` on your MySQL server.
4. Use the SQL code from `sql_for_db.txt` to create the tables in the `bc_database` database.
5. Use the queries from `queries_for_data.txt` to insert sample data into the tables.
6. Open a web browser and visit `http://localhost/bear_chef` to view the website.
